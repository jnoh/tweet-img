# tweet-img

> Tweet an image

## Installation

```
npm install tweet-img
```

## Usage
```
const tweetImg = require('tweet-img');

//const imgData = 'binary imgData';

tweetImg.setup('consumerKey', 'consumerSecret', 'token', 'tokenSecret');
tweetImg.tweet('image/jpeg', imgData);

```
